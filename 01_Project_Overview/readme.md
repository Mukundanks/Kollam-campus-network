#  Building an ultrafast internet backbone for educational institutions in Kollam

## Overview
This section gives a clear idea of what the project is, why is it important and how it will be built etc. Everything in this project is designed against a simulated set of clients' requirement and future growth. This project, as I call *Kollam Regional Metro-Campus Network* (KRMN) is my self proposed project based on real entities and real challenge. The respective names has been used solely to add authenticity to the project, and not for any other purpose. I thereby confirm there is no actual connection between this project and the entities network infrastructure, and whatever I design and implement here is from my own imagination and research knowledge. 


## Why this project?
My personal motivation behind this project is to improve my knowledge with network designs and implementation, and also to push my knowledge beyond CCNA limits. I will be bringing in technologies which are known to me, and those I found while going through different forums. I am using different LLM tools like Gemini, Claude and ChatGPT to act as my mentor and senior network engineer, to test and verify my work. This is to simulate a standard corporate office setup, and not to cheat myself in this process. All the LLMs have been prompted according to this criteria. 

## Scope of this project:
This project brings in various futureproof technologies to the institutions mentioned above. This will potentially improve students' access to the internet and access to the information, thereby improving their knowledge and skill base. Thereby designing and building this real-world network simulation, the institutions who face the same problem can inspire from this and implement it as a part of their commitment to towards students and nation building. 

## Project goals:
The end goal is to design an end to end network infrastructure and document everything, as it is done by any firm. This will be the standard procedure, also on top of that, I will be confident in all the technologies used in this project, and documenting. For me, success looks like finishing this project as a pro in network design and implementation. 

## Tools used:
I will be using Cisco's Packet Tracer and GNS-3 for design and simulation. The servers like DHCP and DNS will be implemented using Windows server(evaluation version for study). Planning to implement network monitoring tools using Docker running on Linux OS. The GNS-3, Win Server and Linux will be running as VM in VMware Workstation Pro.
